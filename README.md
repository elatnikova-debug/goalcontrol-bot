# 🎯 Коуч-Трекер — Telegram Bot

Персональный коуч-трекер и мотиватор в Telegram. Помогает ставить цели, разбивать на этапы, отслеживать прогресс и получать мотивацию.

## Возможности

- 📌 Создание целей с дедлайнами
- 📍 Разбиение целей на этапы с промежуточными сроками
- ⏰ Автоматические напоминания о дедлайнах
- ✅ Отметка выполненных этапов и целей
- 💪 Мотивационные сообщения и похвала
- 📊 Статистика и прогресс (стрики, прогресс-бары)
- 🌅 Утренняя мотивация и вечерние проверки
- ⭐ Монетизация через Telegram Stars (30 дней бесплатно, потом подписка)

## Монетизация

- Первые 30 дней — бесплатно (trial)
- После — подписка за 250 Telegram Stars/месяц (≈ $5)
- Оплата прямо в Telegram, без внешних сервисов
- Автоматические напоминания об окончании подписки/trial

## Быстрый старт

### 1. Создай бота

Напиши [@BotFather](https://t.me/BotFather) в Telegram:
```
/newbot
```
Следуй инструкциям, получи токен.

### 2. Настрой

```bash
git clone <your-repo-url> coach-bot
cd coach-bot
cp .env.example .env
# Открой .env и вставь свой BOT_TOKEN
```

### 3. Запусти локально

```bash
pip install -r requirements.txt
export BOT_TOKEN="твой_токен_здесь"
python main.py
```

### 4. Деплой на Railway (рекомендуется)

1. Загрузи код на GitHub
2. Зайди на [railway.app](https://railway.app)
3. Создай новый проект → Deploy from GitHub Repo
4. Добавь переменную `BOT_TOKEN` в настройках
5. Railway автоматически обнаружит Procfile и запустит бота

### 4 (альт). Деплой через Docker

```bash
docker build -t coach-bot .
docker run -d \
  -e BOT_TOKEN="твой_токен" \
  -v coach_data:/app/data \
  --name coach-bot \
  coach-bot
```

## Переменные окружения

| Переменная | Описание | По умолчанию |
|---|---|---|
| `BOT_TOKEN` | Токен Telegram бота (обязательно) | — |
| `SUBSCRIPTION_PRICE_STARS` | Цена подписки в Stars | `250` |
| `SUBSCRIPTION_DAYS` | Длительность подписки (дни) | `30` |
| `DB_PATH` | Путь к файлу SQLite | `coach_bot.db` |

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | Приветствие и регистрация |
| `/newgoal` | Создать новую цель |
| `/goals` | Все активные цели |
| `/goal_ID` | Подробности о цели |
| `/done` | Отметить этап выполненным |
| `/complete` | Завершить цель |
| `/cancel_goal` | Отменить цель |
| `/today` | План на сегодня |
| `/motivate` | Порция мотивации |
| `/stats` | Статистика и прогресс |
| `/subscribe` | Оформить подписку |
| `/mystatus` | Статус подписки |
| `/help` | Список команд |

## Структура проекта

```
coach-bot/
├── main.py           # Точка входа
├── bot.py            # Handlers, команды, оплата
├── database.py       # SQLite: цели, этапы, подписки, платежи
├── motivation.py     # Мотивационные сообщения
├── scheduler.py      # Планировщик напоминаний
├── requirements.txt  # Зависимости
├── Dockerfile        # Docker-контейнер
├── Procfile          # Для Railway/Heroku
└── .env.example      # Шаблон переменных
```
